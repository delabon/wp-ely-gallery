    Quick Start

    Ely requires Wordpress 5 and above.

    1. Download the zip file.
    2. Extract it.
    3. Login to your Wordpress Dashboard.
    4. Click on Plugins then click on Add New.
    5. After that click on Upload Plugin then click on Choose File.
    6. Select the zip file from the Folder ely-all and click on Install Now. ( File called ely-gallery-wordpress.zip )
    7. Activate the plugin and enjoy the new Awesome Galleries.

    That's it.

    Create Your First Gallery

    1. Login to your Wordpress dashboard.
    2. Create a new post/page.
    3. Click on Add Block or the + button.
    4. Search for Ely Gallery.
    5. Click on the Ely Gallery Icon.

    Congratulation you have just created your first Ely gallery.

    Block Settings

    Design

    Choose between 4 designs:

    1. Justified
    2. Grid
    3. Fullwidth
    4. Masonry

    Responsive Settings

    Set Columns & Front size by device. This way ensures your gallery to look good on all devices.

    Each column contains one image.

    Row Height Option ( Justified Design )

    If you want to make the images height bigger or smaller change this option.

    This option works only with the Justified Design.
